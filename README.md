# Asset-Management
  这是一个公司IT相关设备的资产管理应用，目前主要有<用户管理> <角色管理> <资产管理> 这三大模块。采用前后端完全分离的架构，后端API参考了RESTful风格。
## 前端
  vue + jquery + bootstrap  
  在experiment分支里实现了一个简单的(Post Put Delete)函数并且实践了javascript的继承。
## 后端
  python + flask + SQLAlchemy
