# NCCC多用途系统
  这是在NCCC工作时做的一个多用途系统，主要有<用户管理> <角色管理> <资产管理> <EU-DLR> <JLR报文查询> 这几个模块。采用前后端完全分离的架构，后端API参考了RESTful风格。
## 前端
  vue + jquery + bootstrap  
  在experiment分支里实现了一个简单的(Post Put Delete)函数并且实践了javascript的继承。
## 后端
  python + flask + SQLAlchemy
